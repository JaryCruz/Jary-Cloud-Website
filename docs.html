Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install jary-cli

Npm

$ npm install jary-cli

Yarn

$ yarn install jary-cli
